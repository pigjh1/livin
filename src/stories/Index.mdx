## 🎯 프로젝트 목표

퍼블리셔 경력을 바탕으로 React 프론트엔드 개발자로 전환하기 위해 제작한 포트폴리오입니다.
실제 쇼핑몰 서비스와 동일한 흐름 (상품 목록 → 상세 → 장바구니 → 주문) 을 구현하고,
컴포넌트 설계, 상태관리, API 연동, 최적화까지 프론트엔드 개발의 핵심 역량을 담았습니다.

---

## 🛠 기술 스택

| 분류 | 기술 |
|------|------|
| 프레임워크 | React + Vite + TypeScript |
| 스타일 | Tailwind CSS |
| 상태관리 | Zustand |
| 서버 상태 | TanStack Query (React Query) |
| 라우팅 | React Router v6 |
| 애니메이션 | Framer Motion |
| Mock API | json-server |
| 배포 | Vercel + Serverless Function |
| 컴포넌트 문서화 | Storybook + Chromatic |

---

## 📌 주요 기능

- 상품 목록 조회 / 카테고리 필터 / 검색 (debounce) / 가격순 정렬
- 상품 상세 페이지 (스펙, HTML 상세정보, 구매평, Q&A 탭)
- 장바구니 담기 / 수량 변경 / 삭제
- 주문서 입력 폼 (유효성 검사) → 주문 완료
- 다크모드 / 로딩 스켈레톤 / 페이지 전환 애니메이션 / 토스트 알림

## ⚙️ 기술적 의사결정

### Zustand
Redux 대비 보일러플레이트가 적고, 장바구니처럼 클라이언트 상태관리에 최적화되어 있습니다.

### TanStack Query
`useEffect` + `useState` 방식에서 벗어나, `staleTime` 5분 캐시로 불필요한 API 재호출을 줄였습니다.

### useMemo 최적화
카테고리 필터 + 검색 + 정렬이 복합적으로 동작하는 상품 목록을 `useMemo`로 최적화했습니다.

### Vercel Serverless Function
json-server는 로컬 전용이므로, 배포 환경에서 동일한 REST API 구조를 Serverless Function으로 구현했습니다.
